# 简介
基于Unity开发的独立游戏。玩法功能代码开源。

# 编辑器
Assets\Scene\pve_levels\下的场景文件打开后可以进行关卡编辑。菜单Window/Map Editor(Ctrl/Command + E)打开编辑器
游戏玩法分为探索和战斗两部分，可以在编辑器中以开始探索、开始战斗、结束战斗、继续探索等“事件”进行切换。
